VibeCheck Pro User Guide

Getting Started

Welcome to VibeCheck Pro! This tool helps you analyze IDE files and generate comprehensive reports.

Adding Files

You can add IDE files in two ways:

  • Drag and Drop: Simply drag your .ide files into the file list area
  • Add Files Button: Click the "Add File(s)" button to browse and select files

Note: When adding multiple files, they will be combined into a single report.

Managing Files

  • Remove Selected: Select files and click "Remove Selected" to remove them
  • Clear All: Click "Clear All" to remove all files from the list

Output Settings

Output Directory: Choose where your reports will be saved

Location/Tool Name: Enter the location or tool name for your analysis

Running Analysis

  1. Add your IDE files
  2. Set your output directory
  3. Enter a location/tool name (optional)
  4. Click "Run Analysis"

Progress Tracking

The progress bar shows the current status of your analysis. You can see detailed progress in the status bar at the bottom of the window.

Error Handling

If an error occurs during analysis:

  • You'll see an error message with details
  • You can generate an error report for troubleshooting
  • The error report will be saved to your desktop

Tips

  • Keep your IDE files organized for easier analysis
  • Use descriptive location/tool names for better report organization
  • Check the status bar for important messages and progress updates
